## Getting Started

### Prerequisites
- Node.js 18+
- Docker & Docker Compose

### Quick Start
```bash
# 1. Start database
docker-compose up -d

# 2. Start backend (Terminal 1)
cd backend && npm install && npm run start:dev

# 3. Start frontend (Terminal 2)
cd frontend && npm install && npm run dev
```

**URLs:**
- Frontend: http://localhost:5173
- Backend API: http://localhost:3000

3. Update frontend services for any new endpoints

2. TypeORM will auto-sync in development mode
3. Document schema changes in BACKEND_DOCUMENTATION.md
